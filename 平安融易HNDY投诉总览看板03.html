📋 Excel 导入字段说明

导入的 Excel 文件需包含以下工作表（Sheet），系统会按表名关键字自动匹配。

Sheet①：月度汇总 必需

每行代表一个月的汇总数据，系统据此生成折线图、柱状图和月度表格。

列名（表头）类型必填说明示例值
月份文本/数字★必填写法：1月/ 2月 或 1 / 24月
非诉案量数字★必填当月非诉渠道委案总量8934
非诉口径件数字★必填非诉渠道口径投诉件数10
非诉投诉率小数可自动算= 非诉口径件 / 非诉案量0.001119
金调案量数字★必填当月金调渠道委案总量38227
金调口径件数字★必填金调渠道口径投诉件数42
金调投诉率小数可自动算= 金调口径件 / 金调案量0.001099
总案量数字可自动算= 非诉案量 + 金调案量47161
总口径件数字可自动算= 非诉口径件 + 金调口径件52
总投诉率小数可自动算= 总口径件 / 总案量0.001103
工作日数数字★必填当月工作日天数22

Sheet②：投诉明细 必需

每行代表一条投诉记录。表头在第1行，数据从第2行开始。

列名（表头）类型必填说明有效值
月数字★必填该投诉所属月份1 / 2 / 3 / 4 ...
是否涉监管文本★必填是否涉及监管投诉是/ 否
是否明确涉及三方短信文本★必填是否涉及三方短信是 / 否
内部是否有责文本★必填内部稽核是否有责是 / 否
是否为江苏地区案件文本★必填是否江苏地区是 / 否
撤诉情况文本★必填撤诉结果成功 / 失败 / 空
条线文本★必填业务条线分类（用于第三模块）外访催收 / 债务延期 ...
咨诉类型文本★必填投诉诉点分类（用于第四模块）外访催收 / 费用争议 ...
渠道文本可选区分非诉/金调，无则按投诉等级判断非诉 / 金调
投诉等级文本可选无渠道列时用此判断（一级=非诉）一级投诉 / 二级投诉
是否为融易贷文本可选是否融易贷专项是 / 否

Sheet③：辅助表 可选但推荐

每月的阈值指标、预警红灯数等需手动维护的数据。不导入此表则阈值显示"--"。

列名（表头）类型说明示例
月份数字/文本对应月份4 或 4月
阈值指标数字甲方每月下发的阈值20
预警红灯数数字当月预警红灯数量0

⚠️ 注意事项

1. 表名匹配：系统按关键字匹配，如包含"汇总"会识别为月度汇总表
2. 列名匹配：列名包含关键字即可，如"非诉案量(件)"也能识别
3. 条线/诉点名称：建议使用标准名称（外访催收/债务延期/费用争议/误导还款/代客还款/联络联系人/提醒态度差/提醒频繁/冒充公检法/结清困难/展业误导），不在列表中的自动归入"其他"
4. 阈值指标：必须通过辅助表导入，否则显示为"--"
